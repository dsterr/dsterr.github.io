About This Website

This website was created as a simple, clean space to share resources, activities, and information. It’s built using GitHub Pages, which allows the entire site to be hosted for free and updated easily.

The goal of this site is to provide a clear, accessible hub for anyone who wants to explore the content—whether that’s an activity pack, a project, or a collection of useful materials.

Everything here is designed to be lightweight, fast, and easy to navigate. As the site grows, more pages and resources will be added.

What You’ll Find Here

• A homepage introducing the project • Activity or resource pages • Contact or credit information • Additional sections as the site expands

Why GitHub Pages?

GitHub Pages is ideal for simple websites like this one. It’s reliable, free, and integrates directly with version control—making updates quick and transparent.
